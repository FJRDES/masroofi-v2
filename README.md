# Masroofi V3.0 - المرحلة الأولى

إضافة لوحة الإحصاءات الذكية في الصفحة الرئيسية:
- أكثر تصنيف صرفت عليه هذا الشهر.
- أكثر مستفيد.
- أكثر مكان.
- متوسط الصرف اليومي.
- مقارنة نفس الفترة من الشهر الحالي بالشهر السابق.
- توقع نفاد أو كفاية الميزانية بناءً على معدل الصرف اليومي.

تم الحفاظ على بنية Firebase الحالية دون تغيير مسارات البيانات.
