# Masroofi V3.7

إضافة لوحة الإحصاءات الذكية لدورة الميزانية:
- أكثر تصنيف.
- أكثر مستفيد.
- أكثر مكان.
- متوسط الصرف اليومي.
- أعلى يوم صرف.
- عدد عمليات الدورة ومتوسط العملية.
- أكبر عملية.
- توقع انتهاء الميزانية ومؤشر صحة الدورة.

اللوحة تعتمد على دورة الميزانية الحالية فقط، ولا تغيّر حسابات اليوم أو الأسبوع أو الشهر أو الميزانية.
